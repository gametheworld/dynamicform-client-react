Form

成员 说明 类型 默认值
isUpdate 除创建表单为true,则为否 bool -false
formDefinitionSrc 表单地址 string -URL
submitDataSrc 表单数据提交地址 string -URL
formDataSrc 回填数据地址 string -URL
onSubmit 表单提交方法 function(formData); fomData 为当前表单JSON - -function(formData)
beforeSubmit 表单提交前事件 - -function()
onSuccess 表单提交成功时间，httpstatus：200后执行 - -function(response)
onError 表单提交报错后执行 - -function(error)



onSubmit如果定义了自定Submit，那么默认的beforeSubmit、onSuccess、onError不会执行
