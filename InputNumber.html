type: 元素类型

name: instance name - 唯一

path: 数据结构名称

inputType: 输入框类型- 参考html input type

label: 输入框label

placeholder: 输入框placeholder

textAfter: 文本框后置文字

defaultvalue:默认值

min: 最小值

max: 最大值

disabled:是否禁用组件

hidden: 是否显示，不显示：true，显示：false；默认false

inputStyle: 样式配置

disabled:是否禁用组件

rules: Array ,验证规则

target:联动源的path/name

definitionMap:Object,如果定义 definitionMap，则必须有default字段，例如valueA联动valueB，当definitionMap[valueA]有值，则当前组件的属性被definitionMap[valueA]覆盖，否则使用definitionMap['default']覆盖
