联动赋值

            
                
                {
                    ...任意组件
                    "options":[
                        {
                            "sourceAttr": "freestyle",
                            ...
                        }
                    ],
                    "valueMap" :{
                        "sourceAttr1": "targetpath1",
                        "sourceAttr2": "targetpath2",
                        ...
                    },
                }

这套规则目前只适用于有options的组件

options: 必须有。类型：数组类型。数组里的值必须是对象

valueMap: 存储联动赋值的信息

sourceAttr: 当前options数组里对象的属性

targetpath: 待赋值的目标组件path
