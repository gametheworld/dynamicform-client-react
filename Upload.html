upload

            
                
                {
                    "type" : "upload",
                    "fileType":"text|picture",
                    "urlKey":"imagePath",
                    "name" : "files",
                    "path" : "UploadFile.files",
                    "label" : "上传",
                    "action" : "//jsonplaceholder.typicode.com/posts/"，
                    "hidden":false
                    "disabled":false
                    "rules":[{
                        required: true,
                        message: "请选择上传文件"
                    }]
                    "multiple" : true,
                    "disabled" : false,
                    "showUploadList" : true,
                    "accept" : "image/*,.pdf",
                }

type: 元素类型

fileType: 文件类型（text|picture）

urlKey: 加载的数据里的文件网络地址字段名（用于图片的预览或文件的查看）

e.g.: response = {code:20000, data:{imagePath:"http://xxxx"}}，这里urlKey:imagePath

name: instance name - 唯一

path: 数据结构名称

label: 输入框label

action: 上传文件接口

disabled:是否禁用组件

hidden: 是否显示，不显示：true，显示：false；默认false

rules: [{}],目前只支持控制文件是否必须上传，以及提示信息

multiple:是否多文件上传 ,非必填，默认true

disabled:是否可编辑 ，非必填，默认false

showUploadList:是否显示上传文件列表 ，非必填，默认true

accept:可上传文件类型，非必填,默认“image/*,.pdf”
