type: 元素类型

name: instance name - 唯一

path: 数据结构名称

label: 输入框label

placeholder: 输入框placeholder

textAfter: 文本框后置文字

disabled:是否禁用

Style: 样式配置

rules: Array ,验证规则

valueMap:{"options里item的属性名1":'联动目标组件的path1/name1',"options里item的属性名2":'联动目标组件的path2/name2'}

options: Array 选项数据

textField: 文本字段

valueField:值字段

components: 子集元素集合

disabled:是否禁用组件

hidden: 是否显示，不显示：true，显示：false；默认false
